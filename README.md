# MediaGrabber — Dağıtım Kanalı

Bu repo MediaGrabber eklentisinin **imzalı** dağıtımını barındırır (Zen/Firefox auto-update).
Kaynak kod ayrı (private) repoda; burada yalnızca:

- `updates.json` — Firefox/Zen `update_url` manifesti
- Releases altında imzalı `.xpi` dosyaları

Güncellemeler ana repodaki GitHub Actions tarafından otomatik yayınlanır.
